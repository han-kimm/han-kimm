### *지구에 이로운 제품을 찾는 서비스 **Vegwins***
1인 프로젝트, MERN 스택으로 프론트엔드와 백엔드를 모두 개발하였습니다. 백엔드는 BFF를 빠르게 구축하는 데 목표를 두었습니다.<br/>
프론트엔드는 Axios, 상태관리 라이브러리 없이, Next14를 최대한 활용하여 개발하였습니다.
